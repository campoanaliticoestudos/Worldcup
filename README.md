# Banco de Dados: Campo Analítico

Este repositório contém o conjunto de dados utilizado pelos alunos do **Campo Analítico** para fins de estudos, treinamento e desenvolvimento de modelos preditivos no contexto de futebol.

## Objetivo

O propósito desta base é fornecer dados estruturados para a aplicação prática de técnicas de **Estatística Computacional**, permitindo que alunos realizem desde a captura de eventos até a criação de mapas de gols, testes de hipóteses e simulações de Monte Carlo.

## Conteúdo do Repositório

Os arquivos estão organizados em formato CSV e abrangem diversos confrontos internacionais, sendo essenciais para a construção de um portfólio profissional em análise de dados esportivos.

## Como utilizar

Para processar estes dados, recomenda-se o uso da linguagem **R**, a ferramenta de eleição para a modelagem estatística em nossa metodologia, devido à sua capacidade de integrar dados brutos a modelos preditivos complexos.
